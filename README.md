# 音频测试系统

## 项目结构
```
audio-test-system/
├── packages/
│   ├── server/              # Node.js 后端服务
│   ├── teacher/             # Electron 教师端应用
│   └── student/             # Vue 学生端应用
├── package.json
├── pnpm-workspace.yaml
└── README.md
```

## 开发说明
1. 安装依赖：`pnpm install`
2. 开发模式：`pnpm dev`
3. 构建：`pnpm build`
4. 打包教师端：`pnpm package`
